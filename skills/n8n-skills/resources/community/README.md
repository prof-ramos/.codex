## Installation

Community nodes can be installed via n8n's GUI:

1. Go to **Settings** > **Community Nodes**
2. Click **Install**
3. Enter the package name (e.g., `n8n-nodes-evolution-api`)
4. Click **Install**

> Note: Community nodes are only available on self-hosted n8n instances.
